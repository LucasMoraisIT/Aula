Tipos de Testes

Teste de Caixa Preta

O teste de caixa preta avalia a funcionalidade de um sistema sem olhar para o seu código interno. Os casos de teste são desenvolvidos com base nos requisitos funcionais do sistema.

Teste de Caixa Branca

O teste de caixa branca, também conhecido como teste estrutural, avalia a estrutura interna de um sistema. Os casos de teste são desenvolvidos com base no código fonte do sistema.

Teste de Tarefa

O teste de tarefa verifica se o sistema é capaz de executar tarefas específicas conforme o esperado, muitas vezes focando na experiência do usuário durante a interação com o sistema.

Teste Comportamental

O teste comportamental avalia o comportamento do sistema em diferentes situações e condições. Ele pode incluir testes de desempenho, estresse e carga para garantir que o sistema atenda aos requisitos de desempenho.

Teste de Intertarefas

O teste de intertarefas avalia a integração entre diferentes partes ou módulos de um sistema. Ele verifica se as interações entre essas partes são executadas corretamente e se os dados são transferidos de maneira adequada.

Teste de Sistema

O teste de sistema avalia o sistema como um todo, verificando se todos os componentes integrados funcionam corretamente juntos conforme especificado nos requisitos.
